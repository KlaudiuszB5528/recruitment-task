# recruitment-task
My solution to the recruitment task

## Getting started

- npm

  ```sh
  npm install npm@latest -g
  npm install --global yarn
  ```

### Installation

1. Clone the repo

   ```sh
   https://github.com/KlaudiuszB5528/recruitment-task.git
   ```

2. Install NPM packages

   ```sh
   yarn install
   ```

3. Run
   ```sh
   yarn dev
   ```
